- 限制粒子数量为 20 个

## 技术细节

### 使用的 motion/react 特性
1. `motion.div` - 基础动画容器
2. `animate` 属性 - 持续动画
3. `initial/whileInView/whileHover/whileTap` - 交互状态
4. `useScroll` + `useTransform` - 滚动联动
5. `transition` 配置 - 自定义动画曲线

### 性能优化
- 所有动画使用 transform 和 opacity
- SVG 使用 GPU 加速
- 粒子使用绝对定位避免重排
- 动画时长错开避免同时计算
